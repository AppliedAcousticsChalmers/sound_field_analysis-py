Usage¶

Requirements¶

Obviously, you’ll need Python. The code has been developed and tested on Python 3.7 only. NumPy and SciPy are needed for calculations. .. If you also want to plot the resulting sound fields, you’ll need matplotlib.

Instead of installing all of them separately, you should probably get a Python distribution that already includes everything, e.g. Anaconda.
